Схема клиент-серверного взаимодействия по протоколу HTTP.

Клиент (браузер) отправляет HTTP-запрос на сервер.
Сервер обрабатывает запрос и возвращает HTTP-ответ.
Ответ содержит HTML, CSS и JavaScript, которые браузер отображает пользователю.
Также показаны основные коды состояния: 200, 404 и 500.
